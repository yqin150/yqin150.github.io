原型图工具axure&Marvelapp

我这一周学习的工具主要是axure和MarvelApp这两个原型工具都是以微信为原型来做的设置，这里我主要介绍一下我在学习上面遇到的各种坑点

看到下面的这个图我的第一个坑点就是没有使用原型里面一个强打的工具就是母版，下面的四个切换的按钮因为有四个页面都要用，所以我们在设计网第一个页面的时候可以通过增加母版，从而重复利用下面啊的四个按钮


接下来我想要介绍的就是Axure里面的动态面板，这个面板可以帮助我们实现我们想要的任何切换的功能，看下面的这个图片



方法就是

  • 从工具相里面脱出一个动态面板,双击state，在这个页面里面可以添加任何你想要添加的东西，（注意一点的就是以为你在项目中会有很多的面板，为了让你的面板可以方便以后的管理就是一定要命名）
  • 上面已经完成的差不多了，在我们下一步操作的就是hidden这个面板，然后点击properties添加你想要添加的事件，通常情况下就是show this 动态面板
  • 最后就是点击返回的时候在返回hidden this动态面板

接下来要说的就是Marvel，这个就是简单一点就是
方法就是

  • 任何你想要添加动作的地方
  • 添加一个类似模态框或者是popup的东西就是点击相机，选择Layer an image然后就是设置这个面板的替考位置

  • 还有就是一定要记得添加Save Layer
  • 最后就是选择一个展示的形式就好了（Screen Transition）
